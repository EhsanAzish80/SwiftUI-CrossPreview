# VS Code Extension Development

This directory contains the VS Code extension for SwiftUI CrossPreview.

## Project Structure

- `src/` - TypeScript source files
  - `extension.ts` - Main extension entry point with webview implementation
- `media/` - Static assets (CSS, images, etc.)
  - `preview.css` - Styles for the preview webview
- `parser/` - Tree-sitter based SwiftUI parser (to be implemented)
- `renderer/` - HTML/CSS renderer for SwiftUI components (to be implemented)
- `dist/` - Compiled extension code (generated by esbuild)

## Development

### Build

```bash
npm run build
```

### Watch Mode

```bash
npm run watch
```

### Lint

```bash
npm run lint
```

## Dependencies

- **tree-sitter** - Parser generator tool for building syntax trees
- **tree-sitter-swift** - Swift language grammar for tree-sitter
- **esbuild** - Fast JavaScript bundler for building the extension

## Testing the Extension

1. Open this folder in VS Code
2. Press F5 to launch the Extension Development Host
3. Open a Swift file or use Command Palette: "SwiftUI: Open Preview"
